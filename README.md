
## Frontend E-Commerce Website (startling-babka-729501.netlify.app)
This is a fully responsive frontend e-commerce web application developed using HTML, CSS, and JavaScript. The project replicates the core functionality of a modern fashion shopping platform with structured navigation, real-time filtering, and interactive UI components — all without backend integration.

🚀 Features

🛍 Category & Type-Based Filtering
Products are organized into main categories like Shirts, T-Shirts, and Pants, with additional sub-type filters such as Plain, Printed, Checked, Formal, Jeans, Round Neck, and more. Filtering is handled dynamically using JavaScript class-based logic without page reloads.

🔎 Real-Time Search
Users can search for products using a search bar, and results update instantly based on matching product names.

🧾 Product Modal

- Clicking a product opens a dynamic modal that allows users to:

- Select size

- Choose color

- Adjust quantity

- Add product to cart

🛒 Add to Cart System

- Cart updates instantly

- Cart count changes dynamically

❤️ Wishlist Feature

- Users can add products to a wishlist, which is also stored locally.

🎨 Clean UI & Responsive Layout

- Sidebar navigation

- Product grid layout

- Smooth hover effects

- Clean user experience

- Responsive design for different screen sizes

🛠 Technologies Used

- HTML5

- CSS3

- JavaScript (ES6)

- DOM Manipulation

- LocalStorage for state management
